3. Use Add file > Upload files.
4. Arraste todo o CONTEUDO da pasta extraida para a pagina do GitHub.
5. Espere todos os arquivos terminarem e clique em Commit changes.

   Build command: npm run build
   Deploy command: npm run deploy
